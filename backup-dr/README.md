# Backup + DR Drill

Procedure-driven backup strategy and disaster recovery drills with scripts for PostgreSQL/MySQL and file storage.

- 3-2-1 backups
- RTO: 4h, RPO: 1h
- Automated verification and restore drills
 - Immutable mode: S3 Object Lock (WORM) optional
 - Malware-scanned restores (ClamAV)
